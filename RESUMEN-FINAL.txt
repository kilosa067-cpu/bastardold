💳 PAYPAL - PARA ACTIVAR
─────────────────────────────────────────────────────

1. Ve a https://developer.paypal.com/
2. Crea cuenta de desarrollador
3. Crea una app y copia el Client ID
4. Crea archivo .env:
   VITE_PAYPAL_CLIENT_ID=tu_client_id_aqui
5. En src/sections/Agenda.tsx reemplaza el PayPalButton mock
   por la integración real (instrucciones en README-BASTARD.md)


🚀 DESPLIEGUE EN VERCEL - RESUMEN
─────────────────────────────────────────────────────

1. Reemplaza las imágenes en /public/images/
2. Sube a GitHub:
   git init
   git add .
   git commit -m "Version final"
   git branch -M main
   git remote add origin https://github.com/TU_USUARIO/bastard-barber.git
   git push -u origin main

3. Ve a https://vercel.com
4. Importa el proyecto desde GitHub
5. Framework: Vite
6. Build: npm run build
7. Output: dist
8. Deploy!

⚠️ NOTAS IMPORTANTES
─────────────────────────────────────────────────────

1. Las imágenes actuales son de placeholder (Unsplash)
   Reemplázalas con las tuyas antes de lanzar

2. PayPal está en modo mock (simulado)
   Para activar pagos reales, sigue las instrucciones arriba

3. El admin guarda todo en localStorage del navegador
   Los cambios persisten entre sesiones

4. Para resetear todo, limpia el localStorage del navegador


🐛 SI ALGO FALLA
─────────────────────────────────────────────────────

- Revisa que las imágenes estén en public/images/
- Verifica que el build sea exitoso: npm run build
- Limpia caché del navegador
- Revisa la consola (F12) para errores
